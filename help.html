The menu across the top shows you all that you can manage in the RUCAM system. Below shows the basics of creating, editing and searching records. Some items in this menu have more complex functionality which is described under the tabs above.

The system also allows you to allow cards to enter venues on a certain date for a fixture.

Viewing Records

By clicking on an item in the menu you will be able to view all the records for that item. You will then see another menu, such as the one below, giving you the options to create a new record or search for a particular record.

Creating Records

By clicking on the create new button you will be shown a form, below on the right, where you will be able to enter the data to create a new record. To save the data you must click the create new button located below the form.

Editing Records

To edit a record click the ID number for that record as shown below.

All records have an update page like the one shown below. From here you can edit the record and save the changes by clicking the update button located below the form.

Some items, such as competitors, teams and fixtures, have more functions to do with the record shown below this. For more imformation on this click on the different tabs at the top of this help page.

Searching Records

By clicking on the search button you will shown a form to enter your search criteria. Clicking search will show the results in the table to the left.

There are two search conditions AND and OR. AND means that all of the search fields you enter must exist in the record for it to be shown in the results. OR means that any of the search fields can exist in the record for it to be shown in the results.
